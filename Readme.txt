Steps to run the project:
1)Open the installation files folder, and run the package_installer.bat file. wait for it
to finish downloading the required packages. once it CLOSES ITSELF continue.
2)Start docker on your system, when finished loading move to the next step.
3)Open the Project_start_files folder.
4)Start the Elastic_Loader.bat, and wait for it to finish loading the elastic stack 
5)Now you can run the Orits_Project.bat file to start the project.
